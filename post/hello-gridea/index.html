Hello Gridea

发布于 2022-10-31 | 分类于 Gridea | 3分钟 | 725字数 | 浏览量：

我已经数不清这是第多少次换博客平台了，从一开始的 github pages + Jekyll，没过多久就厌倦于复杂的 gitalk 开启操作和有点难受的路径& tag 管理。后来又尝试了 Hexo 和 Typecho ，没有一个能让我坚持下来的。

或是对一些细节不满意，或是不能实现多个平台的协作，或是不支持仅自己可见的 mode 。零零散散，也未成体系的写出些什么。直到最近才觉得还是需要找一个地方记录一些东西，于是尝试了Gridea 。目前的体验来看还算满意，搭配一些同步工具总算是可以多个平台协作。当然，因为比较懒，更喜欢集成而不是自己做重复性的操作，所以并不是很想使用图床，上次配置的blackBlaze b2+cloudflare 又可以安心吃灰了。虽然可能在访问速度上会带来一些问题，但是写的东西大部分还是给自己看的所以也没那么大关系。

我自己属于比较三天打鱼两天晒网的 type 。可能偶尔更新，大多数时间摸鱼摆烂。其实很多时候想想还是很后悔，因为自己的懒惰在后面的学习工作中造成了本可以避免的麻烦、或者是忘却了本可以记录的精彩瞬间。也因此这是一个小号，没有那种必须在人前表现或者解释什么的必要，有的只是一个有点懒的人某些时刻过剩的分享欲，自然也随性些。不知道是什么驱使我写下这段话，即使我确信很长一段时间里看到这段话的人只有我（&my girlfriend）而已。——真是一个奇怪的人，是吧。

一直以来我对写作输出的态度有种说不出的惶惑，总觉得自己水平不够还要误人子弟，实在是好为人师。但是最近发现经过各种渠道输入的多了，没有经过自己的转化，很多东西转眼就忘。所以 记录 是一件有意义的事情，而 分享 是另一件有意义的事情。从萌新时期到现在成为成熟的老萌新，我一路都是啃着别人分享的知识走来的，现在知道有些坑和弯路，自然也要记录下来分享出去。当然，这两件有意义的事合在一起是否还有意义，那我可不好说。毕竟这里只服务于我自己的需要，写一些胡言乱语，做一个废纸篓。如果真有人能从这里学到东西或者受到启发，那善莫大焉。

行了，今年的更新应该差不多结束了，明年见（bs
